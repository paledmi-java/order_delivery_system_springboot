# Order Delivery API

REST API сервиса доставки товаров, разработанный на Spring Boot.

## Стек

- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Spring Security
- JWT
- Docker
- Swagger (OpenAPI)
- JUnit 5
- Mockito
- Git

## Возможности

- Регистрация и авторизация пользователей
- Получение меню
- Создание и просмотр заказов
- Работа с клиентами
- Валидация входных данных
- Глобальная обработка ошибок
- REST API

## Тестирование

Проект содержит:
- Unit-тесты (JUnit 5, Mockito)
